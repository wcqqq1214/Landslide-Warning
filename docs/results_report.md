# 滑坡预警模型结果汇报

> 2026-06-19 | 代码另附

## 一、项目概述

基于多源监测数据（位移、库水位、降雨、地下水），构建多模型滑坡预警系统，实现从逐日位移预测到四级预警（绿/黄/橙/红）的完整管线。

## 二、数据与方法

- **数据**：8 个位移测点（MJ9、MJ1、MJ3 + 5 个 ATU），日尺度，共 ~1,400 天；附加库水位、降雨、气温、地下水位
- **划分**：前 80% 训练，后 20% 测试（287 天）；全 8 测点纳入所有模型和分析

| 方法 | 用途 |
| --- | --- |
| ConvLSTM | 位移点预测 + P10/P50/P90 预测区间 |
| NGBoost | 四级预警分类（基于测点专属 V0 阈值） |
| SHAP | 特征重要性排序，解释驱动因子 |
| 改进切线角 | 许强 2009 方法，45°/80°/85° 分界 + 持久化判定 |
| 动态 V0 | 基于训练段稳定期统计的测点专属月速率阈值 |

## 三、结果

### 3.1 位移预测（ConvLSTM）

> 📊 **建议放图**：`figures/convlstm/forecast_interval.png`（预测区间图）

8 个测点汇总：

| 指标 | ConvLSTM | 持久性基线 | 提升 |
| --- | ---: | ---: | --- |
| RMSE | 0.313 mm | 0.340 mm | ↓ 8.0% |
| MAE | 0.149 mm | 0.181 mm | ↓ 17.3% |
| P10-P90 覆盖率 | 72.6% | — | 目标 80% |
| 平均区间宽度 | 0.378 mm | — | |

三个重点测点：

| 测点 | RMSE | 基线 RMSE | 提升 |
| --- | ---: | ---: | ---: |
| MJ9 | 0.107 | 0.125 | 13.8% |
| MJ1 | 0.118 | 0.183 | **35.7%** |
| MJ3 | 0.190 | 0.223 | 14.7% |

- 所有测点均优于基线，MJ1 提升最显著
- 区间覆盖率 72.6%，可通过 conformal 校准提升至 80%

### 3.2 预警分类（NGBoost，8 测点）

> 📊 **建议放图**：`figures/ngboost/confusion_matrix.png`（混淆矩阵）

- 全样本：绿 1,131 / 黄 263 / 橙 30 / 红 7（橙/红全部集中在 2017 年 9-10 月）
- 测点专属 V0 阈值（mm/月）：MJ9=7.37，MJ1=20.50，MJ3=18.80，ATU1=18.72，ATU2=14.07，ATU3=27.81，ATU4=12.84，ATU5=32.28
- 测试集：绿 269 / 黄 18 / 橙 0 / 红 0，准确率 95.8%
- 绿色召回率 95.5%，黄色召回率 100%；按完整四级计算的 macro-F1 为 0.432
- ⚠ **诚实说明**：橙/红全部在训练集，测试段仅含绿/黄。准确率反映的是模型对平稳期 vs 轻微加速期的区分能力，不代表对高等级预警的识别能力

### 3.3 特征重要性（SHAP，8 测点）

> 📊 **建议放图**：`figures/shap/shap_reg_summary.png` 和 `figures/shap/shap_cls_summary.png`

| 排名 | 回归 (位移增量) | 分类 (预警等级) |
| --- | --- | --- |
| 1 | disp_rate_lag1 | disp_rate_lag5 |
| 2 | disp_rate_lag2 | GWT_lag5 |
| 3 | disp_accel_lag5 | GWT_lag4 |
| 4 | disp_accel_lag1 | disp_rate_lag4 |
| 5 | disp_accel_lag3 | station_ATU5 |

- **回归**：前 1–2 日位移速率主导次日位移增量，加速度补充转折信息
- **分类**：前 4–5 日位移速率与 GWT 共同主导，但未超过昨日状态持续性基线
- ⚠ 切线角不直接输入 NGBoost，而是在最终融合层进行可审计升级复核

### 3.4 切线角预警（独立体系）

- 8 测点均计算了改进切线角，值域 [-90°, 90°]
- 三级判别：绿色 ≤ 45° < 黄色 < 80° < 橙色 < 85° < 红色
- 配合 5 天窗口 / 3 天命中 的持久化规则抑制单日误报
- 自动等速阶段下，ATU2–ATU4 可触发较多橙/红；其中 ATU4 橙 155 天、红 250 天，须结合专家等速阶段复核
- 已实现 V0 主判、切线角升级复核：绿 885 / 黄 509 / 橙 30 / 红 7
- 246 天由单关键测点切线角触发 `alpha_watch`，最高只升为黄色；本次无多测点/多尺度升级确认

## 四、当前局限

1. **橙/红预警时间聚集**：37 天橙/红全部在 2017 年 9-10 月，按时间切分后测试段无法评价
2. **区间覆盖率偏低**：72.6%，conformal 校准代码已就绪但 `CAL_FRAC=0.0` 未启用
3. **当日状态任务过于容易**：加入速率后 NGBoost AUC=0.994，但持续性基线 AUC=1.000，模型未产生超越基线的提前预警价值
4. **切线角等速阶段待复核**：当前为训练期自动候选段，ATU 测点的高等级天数对参考速率敏感
5. **高等级跨时期不稳定**：5 折中仅 2 折有正类可评，橙/红仍无独立测试事件
