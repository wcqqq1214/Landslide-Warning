## 2. 当前探索性结果

### 2.1 ConvLSTM（8 测点，单次 287 日留出段）

| 指标 | ConvLSTM | 持久性基线 | 判断 |
| --- | ---: | ---: | --- |
| RMSE | 0.318 mm | 0.340 mm | 降低 6.5% |
| MAE | 0.159 mm | 0.181 mm | 降低 12.0% |
| P10-P90 覆盖率 | 73.4% -> 75.2% | 不适用 | 校准后仍低于名义 80% |
| 平均区间宽度 | 0.415 -> 0.432 mm | 不适用 | 校准扩张 0.017 mm |
| 平均 pinball loss | 0.063 -> 0.062 mm | 不适用 | 校准端点仅作诊断 |
| 80% interval score | 1.094 -> 1.068 mm | 不适用 | 校准后改善 2.4% |

仅在该最后留出折中，8 个测点的 RMSE 均低于持久性基线；MJ9/MJ1/MJ3 校准后覆盖率仍只有 47.4%/60.6%/55.4%，其余 ATU 测点为 83.3%-88.9%。

三个连续测试块的 RMSE 为 0.074、0.110 和 0.537 mm，校准后覆盖率为 91.3%、87.9% 和 46.1%。最后时段明显退化，说明固定校准量不能消除跨期漂移。R2/NSE 已输出，但两者在当前定义下相同，且累计位移强趋势使其接近 1，因此不作为主要性能依据。

14 日主块长下，模型相对持久性基线的 RMSE 差值为 -0.022 mm（95% CI -0.029 至 -0.012），MAE 差值为 -0.022 mm（-0.031 至 -0.011）。校准相对原始区间的 interval score 差值为 -0.026 mm（-0.041 至 -0.010），但距名义 80% 覆盖率的绝对偏差变化区间跨 0。7/30 日敏感性分析保持总体误差和 interval score 的方向，但 30 日覆盖率区间扩大到 61.9%-90.9%。这些是固定模型和 `qhat` 的条件性区间，不是未来漂移保证。

三个非重叠滚动折给出相反的稳健性证据：模型 RMSE 为 2.123、0.492、0.318 mm，持久性基线为 0.245、0.120、0.340 mm；逐测点 RMSE 优于基线的数量为 0/8、0/8、8/8。校准覆盖率为 48.8%、94.9%、75.2%，第二折虽然覆盖率升高，但 interval score 从 1.431 恶化至 1.756。模型只在最后一折显示增量价值，不能据此声称跨时期稳定优于基线。

固定协议五种子诊断进一步表明：折 1/2 的 RMSE 和 MAE 均为 0/5 种子超过基线，折 3 为 5/5。折 3 RMSE 为 0.323 +/- 0.008 mm，但预测日增量标准差仅为实际值的 0.164 +/- 0.022，相关性为 -0.048 +/- 0.220。初始化影响误差幅度，但不改变跨折方向；第三折点误差优势不能等同于稳定时序响应。

预注册内层早停后，三折 RMSE 均值降至 1.336、0.239、0.309 mm，相对固定 120 轮分别有 5/5、5/5、4/5 个种子改善；但折 1/2 仍为 0/5 超过持久性基线。第三折覆盖率达到 81.1%，代价是区间宽度由 0.471 增至 0.977 mm、interval score 由 1.037 恶化至 1.248 mm。训练轮数是可纠正的影响因素，但不是当前结构跨时期失效的完整原因。

2x2 有限容量/正则化诊断在折 1/2/3 分别选择 `h16_wd0`、`h08_wd0`、`h16_wd1e4`，第一/二名内层 loss 差值均小于所选配置种子 SD 的 5%。最终三折 RMSE/MAE 正 skill 种子数为 0/5、0/5；0/5、0/5；5/5、4/5，仅折 3 达到双指标多数正 skill。该结果触发预注册停止扩搜规则，不支持继续在当前已查看折上扩大搜索。
