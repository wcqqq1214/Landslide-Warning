## 2. 当前探索性结果

### 2.1 ConvLSTM（8 测点，287 日留出段）

| 指标 | ConvLSTM | 持久性基线 | 判断 |
| --- | ---: | ---: | --- |
| RMSE | 0.313 mm | 0.340 mm | 降低 8.0% |
| MAE | 0.149 mm | 0.181 mm | 降低 17.3% |
| P10-P90 覆盖率 | 72.6% | 不适用 | 低于名义 80% |
| 平均区间宽度 | 0.378 mm | 不适用 | 必须与覆盖率联合解释 |

8 个测点的 RMSE 均低于持久性基线，但覆盖率差异很大：MJ9/MJ1/MJ3 为 37.3%/55.7%/53.3%，其余 ATU 测点为 81.5%-88.5%。

### 2.2 NGBoost 当日四级状态

- 全时段：green 1131、yellow 263、orange 30、red 7。
- 留出段：green 269、yellow 18、orange 0、red 0。
- Accuracy=0.958，完整四分类 macro-F1=0.432，多分类 Brier=0.088。
- Green/yellow recall=0.955/1.000；orange/red 因支持数为 0 而不可评价。

### 2.3 SHAP 当日二分类

- NGBoost：AUC=0.994、PR-AUC=0.792、Brier=0.009、F1=0.766。
- 昨日状态持续性基线：AUC=1.000、PR-AUC=1.000、F1=0.958。
- 5 个扩展窗口测试折仅第 1/5 折同时含正负类；NGBoost AUC=0.914/0.993，持续性基线=0.999/1.000。
- 当前结果支持“能够识别持续状态”，不支持“已经实现未来提前预警”。

## 4. 改进优先级

| 优先级 | 改进项 | 完成判据 |
| --- | --- | --- |
| 已完成 | 实现未来 1/3/7 日 onset 标签和事件提取 | 3/9/21 个正日期，均来自 3 个可预测事件 |
| P0 | 增加独立 onset 事件数据 | 使内层选参和外层评价不由单个事件决定 |
| P0 | 建立按日期的滚动时间验证 | 事件数量足够后，每折训练早于测试，预处理只拟合训练期 |
| P0 | 增加事件级指标和时间块 95% 置信区间 | 输出可追溯 CSV，不以逐行独立 bootstrap |
| P0 | 按有效协议重新调节 NGBoost | 在事件充分前暂停；之后内层选参、外层只评价一次 |
| P1 | ConvLSTM 启用独立校准期 | 同时报告校准前后覆盖率、宽度和 pinball loss |
| P1 | 完成特征组消融与 SHAP 跨折稳定性 | 分离环境量、变化率、累计降雨和运动学量增益 |
| P1 | 完成 V0 与切线角参数敏感性 | 结论不依赖单一参数组合，或明确敏感范围 |
| P2 | 滚动 V0、漂移检测与实时数据接入 | 在新数据上按时间逐步更新且不访问未来数据 |
| P2 | 新时段或跨滑坡确认性验证 | 数据不参与模型和阈值选择 |

在 P0 完成前不更换主模型。任何新增模型对照也必须先获得用户批准。
