## 2. 模型结果

### ConvLSTM（8 测点，287 测试日）

| 指标 | ConvLSTM | 持久性基线 | 判断 |
| --- | ---: | ---: | --- |
| RMSE | 0.313 mm | 0.340 mm | 降低 8.0% |
| MAE | 0.149 mm | 0.181 mm | 降低 17.3% |
| P10-P90 覆盖率 | 72.6% | — | 低于 80% 目标 |
| 平均区间宽度 | 0.378 mm | — | 分位数交叉 0/2296 |

重点测点：

| 测点 | RMSE | 基线 RMSE | RMSE skill | 区间覆盖率 |
| --- | ---: | ---: | ---: | ---: |
| MJ9 | 0.107 | 0.125 | 13.8% | 37.3% |
| MJ1 | 0.118 | 0.183 | 35.7% | 55.7% |
| MJ3 | 0.190 | 0.223 | 14.7% | 53.3% |

### NGBoost 预警分类（8 测点）

- 全样本：绿 1131 / 黄 263 / 橙 30 / 红 7
- 橙/红全部集中在 2017 年 9-10 月（ATU 系列阶跃期）
- 测试集 287 天：绿 269 / 黄 18 / 橙 0 / 红 0；准确率 95.8%
- 绿/黄召回率分别为 95.5%/100%；完整四级 macro-F1=0.432，Brier=0.088
- ⚠ 测试段无橙/红事件，准确率无法反映模型对高等级预警的识别能力

### SHAP（8 测点）

- 回归 top5：`disp_rate_lag1` > `disp_rate_lag2` > `disp_accel_lag5` > `disp_accel_lag1` > `disp_accel_lag3`
- 分类 top5：`disp_rate_lag5` > `GWT_lag5` > `GWT_lag4` > `disp_rate_lag4` > `station_ATU5`
- 二分类留出：AUC=0.994，PR-AUC=0.792，Brier=0.009，F1=0.766
- 持续性基线：AUC/PR-AUC=1.000，F1=0.958，明显优于 NGBoost
- 5 折只有第 1/5 折有正类；NGBoost AUC=0.914/0.993，持续基线=0.999/1.000

## 3. 尚缺的评价指标

| 评价对象 | 缺什么 | 原因 |
| --- | --- | --- |
| 位移点预测 | R2、NSE、滚动窗口误差 | 仅 RMSE/MAE 不足以判断跨时段稳定性 |
| 预测区间 | pinball loss、interval score、校准误差 | 覆盖率与宽度须联合评价 |
| 预警分类 | 概率校准曲线 | PR-AUC 和 Brier 已输出，仍需图形化校准检查 |
| 预警业务效果 | 事件召回率、漏报数、误报天数、提前量 | 预警应评价事件级表现而非逐日标签 |
| 泛化能力 | 按事件分块、可评折的稳健汇总 | 5 折扩展窗口已实现，但 3 折没有正类 |

## 4. 改进优先级

| 优先级 | 改进项 | 预期作用 |
| --- | --- | --- |
| P0 | ConvLSTM 划出校准集 + 启用 conformal 校准 | 覆盖率校准到 ~80% |
| P0 | 预警转折或 3/7 日提前任务 | 避免只复制昨日状态，必须超过持续性基线 |
| P1 | SHAP 候选集补全：切线角、多窗口降雨、速率/加速度 | 完整回答指标筛选问题 |
| P1 | 改善 ATU 系列测点预测和区间覆盖 | ATU 系列位移大但波动也大，是预警关键测点 |
| P1 | NGBoost 输出校准曲线和事件级指标 | PR-AUC/Brier 已实现，继续避免准确率误判 |
| P2 | 指标消融实验 | 量化每类指标的真实增益 |
| P2 | 滚动 V0 + 漂移检测 + 连续预警合并 | 推进到自适应在线预警 |
| P2 | 复核融合层的专家等速阶段 | 当前 V0+切线角融合已实现，需校核 ATU 敏感性 |

P0/P1 完成前不建议更换模型。
