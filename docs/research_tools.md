# 外部科研工具评估与采用记录

> 评估日期：2026-06-21。候选来源为 `modelscope/Awesome-Vibe-Research` 提交 `959f25d` 及各项目当日代码。本文档记录适配判断，不把项目热度、自动化程度或 README 声明视为科研有效性证据。

## 采用原则

1. 外部工具不得修改已冻结的测试折、主要指标、阈值或模型结构。
2. 自动实验工具不得反复查看外层测试结果并据此保留参数或代码。
3. 文献问答、引用核验和图件审查均为辅助流程，最终结论必须回查原文、版本化数据和可复算代码。
4. 工具优先在隔离环境运行，不默认写入项目运行依赖。
5. 安装、接入或复制任何候选项前，必须取得用户明确批准并记录版本。

## 当前优先级

1. 当前模型诊断不依赖任何上述工具，先完成 ConvLSTM 多随机种子稳定性分析。
2. 文献依据需要批量核对时，优先评估 PaperQA2 的隔离运行方案。
3. 结果冻结后，优先评估 `nature-figure` 的只读图件审查流程。
4. 投稿前使用 RefChecker 核验完整参考文献表。
